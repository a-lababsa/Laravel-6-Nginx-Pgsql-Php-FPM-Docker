# Laravel-6-Nginx-Pgsql-Php-FPM-Docker

### Prerequisites

```
docker
docker-compose
makefile
```

### Installing

```
git clone git@github.com:a-lababsa/Laravel-6-Nginx-Pgsql-Php-FPM-Docker.git <projectName>
```

```
cd <projectName>
```

```
make install
```
